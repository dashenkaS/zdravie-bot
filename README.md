# Zdravie Bot (опросный бот для канала Галины Старшиновой)

Проект: телеграм-бот для проведения опросов подписчиков канала о здоровье.

## Что умеет бот
- Проводит опрос как с кнопками-вариантами ответов так и с ответами в свободной форме
- Сохраняет ответы в базу данных SQLite
- После прохождения предлагает записаться на консультацию через встроенную кнопку
- Отправляет частичные ответы после каждого вопроса (данные не теряются если человек бросил опрос)
- Уведомляет администраторов о новых записях на приём

## Запуск проекта

### 1. Установка зависимостей

```bash
# Windows:
py -m venv venv
venv\Scripts\activate
py -m pip install -r requirements.txt
```

### 2. Настройка токена

Создать файл `.env` в корне проекта и вставить туда токен от @BotFather:

```
BOT_TOKEN=ваш_токен_здесь
```

### 3. Запуск бэкенда (в одном терминале)

```bash
py -m uvicorn backend.main:app --reload --port 8000
```

Документация API: http://localhost:8000/docs

### 4. Запуск бота (в другом терминале)

```bash
py -m bot.bot
```

## API эндпоинты

| Метод | Путь | Описание                        |
|-------|------|---------------------------------|
| GET | / | проверка работы сервера         |
| POST | /save_response | сохранить ответ пользователя    |
| GET | /responses | все ответы из БД                |
| GET | /user/{user_id} | ответы конкретного пользователя |
| GET | /stats | агрегированная статистика       |
| DELETE | /user/{user_id} | удалить пользователя            |

## Команды бота

- `/start` — начать опрос или записаться на приём
- `/cancel` — отменить опрос
- `/help` — список команд
- `/admin [пароль]` — стать администратором (пароль = токен бота)
- `/notifications` — включить/отключить уведомления о записях
