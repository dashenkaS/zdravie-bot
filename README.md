# Zdravie Bot (опросный бот для канала Галины Старшиновой)

Учебный проект: телеграм-бот для проведения опросов подписчиков канала "PRO тело со Старшиновой".

## Что умеет бот

- Проводит опрос из 5 вопросов с кнопками-вариантами
- Сохраняет ответы в базу данных SQLite
- После прохождения показывает финальное предложение (приглашение на бесплатную диагностику)
- Отправляет частичные ответы после каждого вопроса (данные не теряются если человек бросил опрос)

## Технический стек

- **Язык:** Python 3.10+
- **Бот:** python-telegram-bot 20.7 (async)
- **Бэкенд:** FastAPI + uvicorn
- **База данных:** SQLite через SQLAlchemy ORM
- **ML:** scikit-learn (KMeans кластеризация)
- **HTTP-клиент бота:** aiohttp

## Запуск проекта

### 1. Установка зависимостей

```bash
python -m venv venv
# Windows:
venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Настройка токена

```bash
# Скопировать .env.example в .env
copy .env.example .env
# Открыть .env и вставить токен бота (получить у @BotFather в Telegram)
```

### 3. Запуск бэкенда (в отдельном терминале)

```bash
uvicorn backend.main:app --reload --port 8000
```

Документация API: http://localhost:8000/docs

### 4. Запуск бота (в другом терминале)

```bash
python -m bot.bot
```

## ML-анализ (машинное обучение)

Модуль `ml_analysis.py` сегментирует подписчиков на 3 группы с помощью алгоритма KMeans:

- **Горячие клиенты** — высокая вовлечённость, готовы к консультации
- **Тёплые читатели** — регулярно читают, но пока не готовы к покупке
- **Случайные гости** — редко заходят, низкая вовлечённость

Чтобы обучить модель, нужно минимум 10 заполненных анкет. После накопления данных отправить POST-запрос на `/ml/train`.
